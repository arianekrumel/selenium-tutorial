Introduction to Web Driver

WebDriver is updated from the original Selenium RC in that it uses native browser API to drive automation instead of injecting JavaScript. WebDriver be used on browser directly or also with Selenium Server run scenarios such as running on other or virtual machines.

Common WebDriver Commands

Locating UI Elements (WebElements) in Java
driver.findElement(By.id("coolestWidgetEvah")); driver.findElements(By.className("cheese")); driver.findElement(By.name("cheese")); driver.findElement(By.cssSelector("#food span.dairy.aged"));
User Input - Filling In Forms
driver.findElement(By.id("submit")).click(); driver.findElement(By.tagName("select")).findElements(By.tagName("option"));
Moving Between Windows and Frames
driver.switchTo().window("windowName"); driver.switchTo().frame("frameName");
Popup Dialogs
driver.switchTo().alert();
Navigation: History and Location
driver.navigate().to("http://www.example.com");
Cookies
driver.get("http://www.example.com"); driver.manage().addCookie(new Cookie("key", "value"));
